10 Essential Algorithms

Binary Search

Search Algorithm

Efficiently finds the position of a target value within a sorted array.

Pseudocode:

function binarySearch(arr, target): left = 0 right = length(arr) - 1 while left ≤ right: mid = floor((left + right) / 2) if arr[mid] == target: return mid else if arr[mid] < target: left = mid + 1 else: right = mid - 1 return -1 // Target not found

Bubble Sort

Sorting Algorithm

Simple comparison-based sorting algorithm that repeatedly steps through the list.

Pseudocode:

procedure bubbleSort(arr): n = length(arr) for i from 0 to n-1: for j from 0 to n-i-2: if arr[j] > arr[j+1]: swap(arr[j], arr[j+1]) return arr

Quick Sort

Sorting Algorithm

Divide-and-conquer algorithm that picks an element as pivot and partitions the array.

Pseudocode:

function quickSort(arr, low, high): if low < high: pi = partition(arr, low, high) quickSort(arr, low, pi-1) quickSort(arr, pi+1, high) function partition(arr, low, high): pivot = arr[high] i = low - 1 for j from low to high-1: if arr[j] < pivot: i = i + 1 swap(arr[i], arr[j]) swap(arr[i+1], arr[high]) return i+1

Dijkstra's Algorithm

Graph Algorithm

Finds the shortest paths between nodes in a graph with non-negative edge weights.

Pseudocode:

function dijkstra(graph, source): dist[source] = 0 create priority queue Q for each vertex v in graph: if v ≠ source: dist[v] = INFINITY add v to Q with priority dist[v] while Q is not empty: u = vertex in Q with min dist[u] remove u from Q for each neighbor v of u: alt = dist[u] + length(u, v) if alt < dist[v]: dist[v] = alt update priority of v in Q return dist

Depth-First Search

Graph Traversal

Explores as far as possible along each branch before backtracking.

Pseudocode:

procedure DFS(graph, vertex): create stack S mark vertex as visited push vertex to S while S is not empty: v = pop from S process vertex v for each neighbor u of v: if u is not visited: mark u as visited push u to S

Breadth-First Search

Graph Traversal

Explores neighbor nodes at the present depth before moving to nodes at the next depth level.

Pseudocode:

procedure BFS(graph, start): create queue Q mark start as visited enqueue start into Q while Q is not empty: v = dequeue from Q process vertex v for each neighbor u of v: if u is not visited: mark u as visited enqueue u into Q

Merge Sort

Sorting Algorithm

Divide-and-conquer algorithm that divides input array into two halves, sorts them, and merges.

Pseudocode:

function mergeSort(arr): if length(arr) ≤ 1: return arr mid = length(arr) / 2 left = mergeSort(arr[0..mid-1]) right = mergeSort(arr[mid..end]) return merge(left, right) function merge(left, right): result = [] i = 0, j = 0 while i < length(left) and j < length(right): if left[i] ≤ right[j]: result.append(left[i]) i = i + 1 else: result.append(right[j]) j = j + 1 append remaining elements of left or right to result return result

Kruskal's Algorithm

Graph Algorithm

Finds a minimum spanning tree for a weighted undirected graph.

Pseudocode:

function kruskal(graph): result = [] // stores the MST sort all edges in graph by weight create disjoint set for each vertex for each edge (u, v, weight) in sorted edges: if findSet(u) ≠ findSet(v): add edge (u, v) to result unionSets(u, v) return result

Knapsack Problem

Dynamic Programming

Selects items with maximum total value, given weight constraints.

Pseudocode:

function knapsack(weights, values, capacity): n = length(weights) create DP table of size (n+1) x (capacity+1) for i from 0 to n: for w from 0 to capacity: if i==0 or w==0: DP[i][w] = 0 else if weights[i-1] ≤ w: DP[i][w] = max(values[i-1] + DP[i-1][w-weights[i-1]], DP[i-1][w]) else: DP[i][w] = DP[i-1][w] return DP[n][capacity]

Floyd-Warshall

Graph Algorithm

Finds shortest paths between all pairs of vertices in a weighted graph.

Pseudocode:

procedure floydWarshall(graph): dist = graph // initialize with direct distances for k from 1 to V: for i from 1 to V: for j from 1 to V: if dist[i][j] > dist[i][k] + dist[k][j]: dist[i][j] = dist[i][k] + dist[k][j] return dist

Array Implementation

Arrays are collections of elements identified by index or key. Here are implementations in C++ and Python:

// C++ Array Implementation #include <iostream> using namespace std; class Array { private: int *arr; int capacity; int size; public: // Constructor Array(int cap) { capacity = cap; size = 0; arr = new int[capacity]; } // Destructor ~Array() { delete[] arr; } // Insert element at index bool insert(int index, int value) { if (size >= capacity) { cout << "Array is full!" << endl; return false; } if (index < 0 || index > size) { cout << "Invalid index!" << endl; return false; } for (int i = size; i > index; i--) { arr[i] = arr[i-1]; } arr[index] = value; size++; return true; } // Remove element at index bool remove(int index) { if (index < 0 || index >= size) { cout << "Invalid index!" << endl; return false; } for (int i = index; i < size-1; i++) { arr[i] = arr[i+1]; } size--; return true; } // Get element at index int get(int index) { if (index < 0 || index >= size) { cout << "Invalid index!" << endl; return -1; } return arr[index]; } // Update element at index bool update(int index, int value) { if (index < 0 || index >= size) { cout << "Invalid index!" << endl; return false; } arr[index] = value; return true; } // Get current size int getSize() { return size; } // Check if empty bool isEmpty() { return size == 0; } // Display all elements void display() { if (isEmpty()) { cout << "Array is empty!" << endl; return; } cout << "Array elements: "; for (int i = 0; i < size; i++) { cout << arr[i] << " "; } cout << endl; } }; // Example usage int main() { Array myArray(10); myArray.insert(0, 5); myArray.insert(1, 10); myArray.insert(2, 15); myArray.insert(1, 7); myArray.display(); cout << "Element at index 2: " << myArray.get(2) << endl; myArray.update(1, 12); myArray.display(); myArray.remove(0); myArray.display(); cout << "Current size: " << myArray.getSize() << endl; cout << "Is empty: " << (myArray.isEmpty() ? "Yes" : "No") << endl; return 0; }
# Python Array Implementation class Array: def __init__(self, capacity): self.capacity = capacity self.size = 0 self.arr = [None] * capacity def insert(self, index, value): if self.size >= self.capacity: print("Array is full!") return False if index < 0 or index > self.size: print("Invalid index!") return False for i in range(self.size, index, -1): self.arr[i] = self.arr[i-1] self.arr[index] = value self.size += 1 return True def remove(self, index): if index < 0 or index >= self.size: print("Invalid index!") return False for i in range(index, self.size-1): self.arr[i] = self.arr[i+1] self.size -= 1 return True def get(self, index): if index < 0 or index >= self.size: print("Invalid index!") return None return self.arr[index] def update(self, index, value): if index < 0 or index >= self.size: print("Invalid index!") return False self.arr[index] = value return True def get_size(self): return self.size def is_empty(self): return self.size == 0 def display(self): if self.is_empty(): print("Array is empty!") return print("Array elements:", end=" ") for i in range(self.size): print(self.arr[i], end=" ") print() # Example usage if __name__ == "__main__": my_array = Array(10) my_array.insert(0, 5) my_array.insert(1, 10) my_array.insert(2, 15) my_array.insert(1, 7) my_array.display() print("Element at index 2:", my_array.get(2)) my_array.update(1, 12) my_array.display() my_array.remove(0) my_array.display() print("Current size:", my_array.get_size()) print("Is empty:", "Yes" if my_array.is_empty() else "No")

Stack Implementation

Stacks are LIFO data structures with infix, prefix, and postfix expression conversion.

// C++ Stack Implementation #include <iostream> #include <stack> using namespace std; class ExpressionConverter { public: string infixToPostfix(string infix) { stack<char> s; string postfix; for (char c : infix) { if (isalnum(c)) { postfix += c; } else if (c == '(') { s.push(c); } else if (c == ')') { while (!s.empty() && s.top() != '(') { postfix += s.top(); s.pop(); } s.pop(); } else { while (!s.empty() && precedence(s.top()) >= precedence(c)) { postfix += s.top(); s.pop(); } s.push(c); } } while (!s.empty()) { postfix += s.top(); s.pop(); } return postfix; } private: int precedence(char op) { if (op == '+' || op == '-') return 1; if (op == '*' || op == '/') return 2; if (op == '^') return 3; return 0; } }; // Example usage int main() { ExpressionConverter converter; string infix = "a+b*(c^d-e)"; cout << "Infix: " << infix << endl; cout << "Postfix: " << converter.infixToPostfix(infix) << endl; return 0; }
# Python Stack Implementation class ExpressionConverter: def infix_to_postfix(self, infix): stack = [] postfix = [] for c in infix: if c.isalnum(): postfix.append(c) elif c == '(': stack.append(c) elif c == ')': while stack and stack[-1] != '(': postfix.append(stack.pop()) stack.pop() else: while stack and self.precedence(stack[-1]) >= self.precedence(c): postfix.append(stack.pop()) stack.append(c) while stack: postfix.append(stack.pop()) return ''.join(postfix) def precedence(self, op): if op in '+-': return 1 if op in '*/': return 2 if op == '^': return 3 return 0 # Example usage if __name__ == "__main__": converter = ExpressionConverter() infix = "a+b*(c^d-e)" print("Infix:", infix) print("Postfix:", converter.infix_to_postfix(infix))

Queue Implementation

Queues are FIFO data structures with basic operations.

// C++ Queue Implementation #include <iostream> using namespace std; class Queue { private: int *arr; int front, rear, capacity, size; public: Queue(int cap) { capacity = cap; arr = new int[capacity]; front = rear = -1; size = 0; } bool isEmpty() { return size == 0; } bool isFull() { return size == capacity; } void enqueue(int value) { if (isFull()) { cout << "Queue is full!" << endl; return; } if (isEmpty()) front = 0; rear = (rear + 1) % capacity; arr[rear] = value; size++; } int dequeue() { if (isEmpty()) { cout << "Queue is empty!" << endl; return -1; } int value = arr[front]; if (front == rear) { front = rear = -1; } else { front = (front + 1) % capacity; } size--; return value; } void display() { if (isEmpty()) { cout << "Queue is empty!" << endl; return; } cout << "Queue: "; for (int i = 0; i < size; i++) { int index = (front + i) % capacity; cout << arr[index] << " "; } cout << endl; } }; // Example usage int main() { Queue q(5); q.enqueue(10); q.enqueue(20); q.enqueue(30); q.display(); cout << "Dequeued: " << q.dequeue() << endl; q.display(); return 0; }
# Python Queue Implementation class Queue: def __init__(self, capacity): self.capacity = capacity self.arr = [None] * capacity self.front = self.rear = -1 self.size = 0 def is_empty(self): return self.size == 0 def is_full(self): return self.size == self.capacity def enqueue(self, value): if self.is_full(): print("Queue is full!") return if self.is_empty(): self.front = 0 self.rear = (self.rear + 1) % self.capacity self.arr[self.rear] = value self.size += 1 def dequeue(self): if self.is_empty(): print("Queue is empty!") return -1 value = self.arr[self.front] if self.front == self.rear: self.front = self.rear = -1 else: self.front = (self.front + 1) % self.capacity self.size -= 1 return value def display(self): if self.is_empty(): print("Queue is empty!") return print("Queue:", end=" ") for i in range(self.size): index = (self.front + i) % self.capacity print(self.arr[index], end=" ") print() # Example usage if __name__ == "__main__": q = Queue(5) q.enqueue(10) q.enqueue(20) q.enqueue(30) q.display() print("Dequeued:", q.dequeue()) q.display()

Tree Implementation

Binary tree with inorder, preorder, and postorder traversals.

// C++ Tree Implementation #include <iostream> using namespace std; class TreeNode { public: int data; TreeNode* left; TreeNode* right; TreeNode(int value) { data = value; left = right = nullptr; } }; class BinaryTree { private: TreeNode* root; void inorder(TreeNode* node) { if (!node) return; inorder(node->left); cout << node->data << " "; inorder(node->right); } void preorder(TreeNode* node) { if (!node) return; cout << node->data << " "; preorder(node->left); preorder(node->right); } void postorder(TreeNode* node) { if (!node) return; postorder(node->left); postorder(node->right); cout << node->data << " "; } public: BinaryTree() { root = nullptr; } void setRoot(TreeNode* node) { root = node; } void inorderTraversal() { cout << "Inorder: "; inorder(root); cout << endl; } void preorderTraversal() { cout << "Preorder: "; preorder(root); cout << endl; } void postorderTraversal() { cout << "Postorder: "; postorder(root); cout << endl; } }; // Example usage int main() { BinaryTree tree; TreeNode* root = new TreeNode(1); root->left = new TreeNode(2); root->right = new TreeNode(3); root->left->left = new TreeNode(4); root->left->right = new TreeNode(5); tree.setRoot(root); tree.inorderTraversal(); tree.preorderTraversal(); tree.postorderTraversal(); return 0; }
# Python Tree Implementation class TreeNode: def __init__(self, value): self.data = value self.left = None self.right = None class BinaryTree: def __init__(self): self.root = None def set_root(self, node): self.root = node def inorder(self, node): if not node: return self.inorder(node.left) print(node.data, end=" ") self.inorder(node.right) def preorder(self, node): if not node: return print(node.data, end=" ") self.preorder(node.left) self.preorder(node.right) def postorder(self, node): if not node: return self.postorder(node.left) self.postorder(node.right) print(node.data, end=" ") def inorder_traversal(self): print("Inorder:", end=" ") self.inorder(self.root) print() def preorder_traversal(self): print("Preorder:", end=" ") self.preorder(self.root) print() def postorder_traversal(self): print("Postorder:", end=" ") self.postorder(self.root) print() # Example usage if __name__ == "__main__": tree = BinaryTree() root = TreeNode(1) root.left = TreeNode(2) root.right = TreeNode(3) root.left.left = TreeNode(4) root.left.right = TreeNode(5) tree.set_root(root) tree.inorder_traversal() tree.preorder_traversal() tree.postorder_traversal()
